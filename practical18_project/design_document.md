# Design Document: Genshin Game Project

## 1. Цель проекта
Создание текстовой игры с каталогом персонажей по мотивам Genshin Impact. Проект объединяет командный интерфейс, графический интерфейс и систему сохранения данных.

## 2. Архитектура
- **core/** – бизнес-логика и модели персонажей.
- **gui/** – графический интерфейс PyQt5.
- **utils/** – вспомогательные функции (загрузка картинок, сохранение/загрузка данных).
- **resources/** – локальные изображения персонажей.
- **tests/** – модульные тесты.

## 3. Компоненты

### 3.1 Каталог персонажей
- Хранение данных в классе `Character`.
- Подгрузка изображений с API или локальных файлов.
- Графическая панель с карточками персонажей (PyQt5).

### 3.2 Текстовая игра
- Выбор персонажа.
- Покроковый игровой цикл.
- Выполнение действий персонажами.
- История ходов и вывод в текстовое окно.
- Сохранение и загрузка состояния игры.

### 3.3 Постоянство данных
- Файл `save_data.json` хранит персонажей и состояние игры.
- Автоматическая загрузка и сохранение.
- Кеширование изображений для уменьшения запросов к API.

### 3.4 Графический интерфейс
- Основное окно `MainWindow` объединяет список персонажей и игровое поле.
- `CharacterCard` отображает портрет, имя, элемент и редкость.
- `GameUI` показывает текстовый вывод игры и кнопки управления.

## 4. API
Использован пример: `https://genshin.jmp.blue/`
- Парсинг JSON.
- Скачивание изображений в `resources/characters`.

## 5. Технологии
- Python 3.10+
- PyQt5
- requests
- unittest (для тестирования)
